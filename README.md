I am a Web Developer and a Building Construction Engineer
---------------------------------------------------------

I have been coding for more than 5 years now. I started off with Html and CSS and subsequently went into Java and C#. The pursuit of a career in building constructions could not let me enhance my coding knowledge since I wasn't giving it enough time. My love and desire to be proficient in coding could not let it go since I would always squeeze out time amidst my busy schedules, to do some coding. For the past 3 years now, I've decided to dedicate more time into coding and build it into a career rather than a hobby. I found JavaScript to be efficient for me and have since expanded into both the front end and backend as a full stack developer.

* 🌍  I'm based in Nigeria
